Java Quiz

  1. What is Java?
    1. Operating System
    2. Programming Language
    3. Database
    4. Web Browser

    Answer: b) Programming Language

  2. Which keyword is used to create an object in Java?
    1. class
    2. object
    3. new
    4. create

    Answer: c) new

  3. Which method is the entry point of Java program?
    1. start()
    2. run()
    3. main()
    4. execute()

    Answer: c) main()
